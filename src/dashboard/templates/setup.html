Step 3 — Browser Setup

        
Launch Chrome or Brave with remote debugging enabled so the bot can control it.

      

      

        
Run this command in your terminal:

        

          
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222 --user-data-dir=/tmp/chrome-debug

          
Using Brave instead?

          
/Applications/Brave\ Browser.app/Contents/MacOS/Brave\ Browser --remote-debugging-port=9222 --user-data-dir=/tmp/chrome-debug

        

        

          
chrome.exe --remote-debugging-port=9222 --user-data-dir=%TEMP%\chrome-debug

          
Using Brave instead?

          
brave.exe --remote-debugging-port=9222 --user-data-dir=%TEMP%\chrome-debug

        

        

          
google-chrome --remote-debugging-port=9222 --user-data-dir=/tmp/chrome-debug

          
Using Brave instead?

          
brave-browser --remote-debugging-port=9222 --user-data-dir=/tmp/chrome-debug
